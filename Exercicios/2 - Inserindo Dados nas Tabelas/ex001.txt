Agora que você criou as tabelas para o banco de dados da biblioteca, é necessário inserir dados para preencher essas tabelas. Você deve inserir informações sobre livros, autores, usuários e empréstimos de acordo com as especificações abaixo:

Inserir Livros:
Você deve inserir informações sobre pelo menos cinco livros diferentes na tabela books. Para cada livro, inclua o título, ano de publicação, nome do autor e quantidade de cópias disponíveis.

Inserir Autores:
Insira informações sobre pelo menos três autores diferentes na tabela authors. Para cada autor, inclua o nome, sobrenome, data de nascimento e nacionalidade.

Inserir Usuários:
Insira informações sobre pelo menos três usuários diferentes na tabela users. Para cada usuário, inclua o nome, e-mail, telefone e endereço.

Realizar Empréstimos:
Simule pelo menos três empréstimos na tabela loans. Para cada empréstimo, selecione um livro da tabela books, um usuário da tabela users e defina a data de empréstimo e a data de retorno prevista.

Lembre-se de seguir as restrições de integridade referencial e assegurar que os dados inseridos estejam consistentes e de acordo com as definições das tabelas. Após inserir os dados, você pode realizar consultas para verificar se os registros foram inseridos corretamente.
